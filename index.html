Docker commands

List images:

docker images
docker image ls

Download images:

docker pull [image]
docker pull [image]:[version]

Delete images:

docker image rm [image]
docker image rm [image]:[version]

Create container:

docker create [container-name]
docker container create [container-name]

it returns an id like a783e16ff5a5c22eeca74a85f7af4300408cdc2fb9cfd9236f33f3be0ffd55bb

Run container:

docker start [id-container]
docker start [unique-id]

Watch run containers:

docker ps

Stop containers running:

docker stop [id-container]
docker stop a783e16ff5a5

Watch container even if its not running:

docker ps -a

Delete container:

docker rm [name | id-container]
docker rm busy_goodall

Create container assigning name:

docker create --name [name] [base-project]
docker create --name mymysql mysql

Listen container logs:

docker logs --follow [container-name]
docker logs --follow mymysql

Download image, create container and run without show logs:

docker run --name mymysql -p 3000:3306 -d mysql
